Toolbox

Data management, parsing and text analysis

By Summer Institute of Linguistics, Inc.

Homepage


- Toolbox -

Features

  • sorting
  • multiple views of the same database
  • filtering to show subsets of a database
  • a user-definable interlinear text generation system which uses a morphological parser and lexicon to generate annotated text
  • interlinear text can be exported in a form suitable for use in linguistic papers
  • export capabilities that can be used to produce a publishable dictionary from a dictionary database

Nociones Principales

Distinguir Proyecto vs. Archivos


Un proyecto es el conjunto
  • de archivos de configuración (contenidos en la carpeta « Settings » dentro de la carpeta del proyecto)
    • Y
  • Los archivos de datos que pueden ser interpretados por el proyecto (lógicamente guardados en la misma carpeta)
